Instant mobile-friendly CSS framework for prototyping.

---

## Installation

### Using _npm_

```bash
$ npm install pcss-framework
```

### Using _CDN_

```html
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/pcss-framework@1.0.1/dist/pcss.min.css"
/>
```
