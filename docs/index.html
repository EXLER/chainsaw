Instant mobile-friendly CSS framework for prototyping.

Usage

Simply include the stylesheet in your HTML files.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/pcss-framework@1.0.1/dist/pcss.min.css" />

Features

  • Responsive
  • Tiny (~6KB)
  • Easy to override
  • Just apply and beautify the page!
